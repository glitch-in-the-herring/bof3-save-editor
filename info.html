Breath of Fire III Save Editor

About

This tool allows you to edit Breath of Fire III save files inside a PS1 memory card file.

Info

There are some important things to know when using this tool:

  • This tool accepts most memory card formats used by emulators (.mcd, .mcr, etc).
  • This tool does not permanently store a copy of your memory card file.
  • Stats are clipped if they exceed the byte range. For example, giving a character 1000 PWR works,
    but trying to give a character 100000 PWR will not work, as that exceeds the 16-bit unsigned integer
    limit, which is 65535.
  • Current HP/AP is the amount of HP/AP the character currently has. Current max HP/AP is the current maximum
    HP/AP the character can have. Current max HP is affected when a character dies mid-battle and is not
    revived, using certain healing items, and not drinking in the Desert of Death. Current max AP is not
    affected by anything to my knowledge. True max HP/AP is the value which the current HP/AP will recover to
    after being afflicted by a non-permanent change, such as the ones mentioned above affecting current max HP.
    Levelling up and using life/magic shards are the only way to increase true maximum HP/AP.
  • A "base" stat is a stat before it is modified by other factors, such as equipment. The reason why I'm showing
    base stats instead of "net" stat is because the game will automatically override net stats calculations, which
    makes that kinda useless in the save file? I think? For example, if Ryu's base PWR is 250 and is equipped with the
    GooKingSword (180 PWR), and assuming nothing else is modifying his power at the moment, his net PWR will be 250 + 180,
    or 430. The values "250" and "430" appear in the save data, but trying to change "430" will be useless, since the game
    will recalculate that upon loading the save file.
  • This editor allows limitations in the game regarding equipability to be bypassed. For example, you can
    equip a LifeArmor as a shield on Nina, or give the KaiserBreath ability to Rei. However, there may be glitches
    especially with the skills, so watch out for that
  • When using the party member, if you do not have access to a party member (eg. Teepo after an event, adult Ryu before an event)
    you cannot add them to your party. It will break the game.
  • If you want some more information on the statistics and resistances, check out ZC_Liu's Breath of Fire III Secrets FAQ
