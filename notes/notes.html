2021/11/02 - 2021/11/??: On Accession

Emulators used: PCSX-Redux, RetroArch (PCSX-ReARMed core), RetroArch (Duckstation core)
Accession is an ability normally only accessible to Ryu. It lets him transform into
a dragon. However, it turns out that other characters can also use Accession, with a few
catches:

Usability

Certain dragon forms don't work on certain emulators. PCSX-ReARMed seems to crash with most dragon
forms, except for Kaiser (needs more testing). DuckStation and PCSX-Redux will not crash with all
dragon forms (needs more testing). However, there will be plenty of graphical glitches, such as the sprite
not rendering properly, or Ryu's sprite being changed into the dragon's sprite, even when he hasn't used
Accession.

Precedence

Two or more characters can use Accession in the same turn, but not using the PCSX-ReARMed core.
If there are two or more characters using Accession, then the dragon forms they transform into depends
on whoever uses Accession last. For example, if Rei's turn is first, and he uses Accession to transform into
a Whelp, but on the same turn, Momo uses Accession to transform into a Kaiser dragon, then both Rei and Momo
will become Kaiser dragons.

Animations and Sounds

Post-timeskip Rei gains the Weretiger ability, giving him animation frames for transformation
abilities. The Weretiger transformation animation is the animation Rei uses when he uses the
Accession ability. For a few frames, you can see the Weretiger before the black sphere effect.
